Laravel 5.8, message app sample

***
### setup

* install Laravel

php composer.phar create-project --prefer-dist laravel/laravel lara58a "5.8.*"

* helper:

php composer.phar require laravelcollective/html "5.8.*"

* .env.local を参考に、 .env 修正下さい

***
### start

php artisan serve
